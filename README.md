## Embedding a deck

Each deck is a self-contained HTML file. To embed one in a blog post, use a responsive iframe:

```html
<div style="position:relative;padding-bottom:56.25%;height:0;overflow:hidden;max-width:100%;">
  <iframe
    src="https://zytelabs.github.io/community-meetups/meetup-1/"
    title="Zyte Community Meetup #1 slides"
    style="position:absolute;top:0;left:0;width:100%;height:100%;border:0;"
    allowfullscreen>
  </iframe>
</div>
```

Navigate the deck with the arrow keys or by clicking the left and right edges.
